About Us

Soulquest is an interactive Learning Platform Powered by ChatGPT is a web-based application that enhances student learning by generating personalized educational content. The platform allows users to create topics and receive questions, feedback, and study materials tailored to their needs. Using ChatGPT, it provides real-time responses based on user performance, making education interactive and dynamic. Built with Flask and SQLAlchemy, Soulquest offers a user-friendly experience with secure login, topic management, and progress tracking.
